# Android-FFmpeg-with-xH264-Build-Script

Build Script for FFmpeg with H264 on Mac

### 文件存放目录结构

```shell
.
├── shell
│   └── armeabi-v7a
│       ├── build_with_x264_armeabi_v7a.sh
│       └── build_x264.sh
└── source
    ├── ffmpeg   -->ffmpeg源码目录
    ├── output   
    │   ├── ffmpeg
    │   └── libx264
    └── x264     -->h264源码目录
```
